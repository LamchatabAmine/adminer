# ADMINER 

### Package used 
- [https://github.com/onecentlin/laravel-adminer](https://github.com/onecentlin/laravel-adminer)


### Run app

```bash
    composer install
```

```bash
    npm install
```

```bash
    cp .env.example .env
```

```bash
    php artisan migrate
```


```bash
    php artisan serve
```

### Access adminer

- Open URL in web browser

```bash
    http://127.0.0.1:8000/adminer
```
